# Windows Install Rsync

Updated version based on https://scicomp.aalto.fi/scicomp/rsynconwindows/

The library versions it had referenced had become very outdated and were no longer available on https://repo.msys2.org/msys/x86_64/

# Usage

Simply download and run within Git-Bash

# Example

Example run within Git-Bash:
```
curl https://raw.githubusercontent.com/kennethburgener/win-install-rsync/refs/heads/main/install-rsync.sh -o install-rsync.sh
bash install-rsync.sh
```
